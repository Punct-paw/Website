# Pawan Bhattarai's Personal Website

A clean, responsive personal portfolio website showcasing my profile as a Structural Engineer. Built with HTML, CSS, JavaScript, and Python (via Brython for animations), this site highlights my education, career, publications, projects, blogs, discoveries, achievements, and contact information.
# Features

Responsive Design: Mobile-friendly layout with hamburger menu for smaller screens.
Sections:

Profile: Introduction with photo, bio, social links, and downloadable CV.
Publications: List of research papers and theses with links.
Projects: GitHub-linked projects with descriptions.
Blogs: Placeholder for future blog posts.
Discoveries: Curated interesting links.
Achievements: Licenses, certifications, awards, and memberships.
Contact: Email and social profiles.


Animations: Decorative moving colored balls around the profile photo, implemented in Python using Brython (runs in-browser).
Accessibility: Semantic HTML, ARIA attributes, and alt text for images.
SEO-Friendly: Proper meta tags, headings, and structured content.
