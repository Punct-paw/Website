# Personal Website
A clean, responsive personal portfolio website showcasing my profile. Built with HTML, CSS, JavaScript, and Python (via Brython for animations), this site highlights my education, career, publications, projects, blogs, discoveries, achievements, and contact information.

---
## Features

- Responsive Design: Mobile-friendly layout with hamburger menu for smaller screens.
- Sections: Profile, Publications, Projects, Blogs, Discoveries, Achievements, Contacts
- Animations: Decorative moving colored balls around the profile photo, implemented in Python using Brython (runs in-browser).
- Accessibility: Semantic HTML, ARIA attributes, and alt text for images.
- SEO-Friendly: Proper meta tags, headings, and structured content.
